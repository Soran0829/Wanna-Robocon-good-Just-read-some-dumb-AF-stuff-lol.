# パソコンのパーツ説明

## CPU (中央演算処理装置)
CPUは「パソコンの頭脳」に例えられ、コンピュータ全体の計算処理や制御を行います。性能が高いほど、複雑な処理を高速に実行できます。代表的なメーカーはIntel (Core iシリーズ) とAMD (Ryzenシリーズ) です。

## メモリ (RAM)
メモリは「作業机の広さ」に例えられます。CPUが処理するデータを一時的に保存する場所で、容量が大きいほど多くのアプリケーションを同時に快適に動かすことができます。

## ストレージ (SSD / HDD)
ストレージは「本棚や引き出し」に例えられ、OSやアプリケーション、写真などのデータを長期間保存する場所です。
*   **SSD (ソリッドステートドライブ):** 高速な読み書きが特徴。OSやよく使うアプリの起動が速くなります。
*   **HDD (ハードディスクドライブ):** 大容量で安価なのが特徴。写真や動画など、大容量データの保存に向いています。

## GPU (グラフィックスプロセッシングユニット)
GPUは、映像や画像を専門に処理するパーツで、「グラフィックボード」や「ビデオカード」とも呼ばれます。3Dゲームや動画編集、AI開発など、高い描画性能が求められる作業で重要になります。

## マザーボード
マザーボードは、CPUやメモリ、ストレージなど、パソコンのあらゆるパーツを接続するための基盤です。各パーツ間のデータのやり取りを仲介する重要な役割を担います。

## 電源ユニット (PSU)
電源ユニットは、コンセントから供給される交流電力を、パソコンの各パーツが使用する直流電力に変換して供給する装置です。各パーツの消費電力の合計よりも余裕のある容量を選ぶことが安定動作の鍵となります。

## 【2024年〜2025年】エンジニア向け おすすめPCスペック

用途によって求められるスペックは異なりますが、近年の開発環境の要求スペック向上を考慮した推奨構成です。

### Web・モバイルアプリ開発向け

*   **CPU:** Intel Core i5 / AMD Ryzen 5 (最新世代) 以上
*   **メモリ:** 16GB以上 (32GBを推奨)
    *   複数の開発ツール、ブラウザ、エミュレータを同時に動かすため、32GBあると快適です。
*   **ストレージ:** 512GB以上のNVMe SSD (1TBを推奨)
    *   OSや開発ツールの起動を高速化するため、SSDは必須です。
*   **GPU:** CPU内蔵グラフィックスで十分な場合が多いです。

### AI・機械学習・データサイエンス向け

*   **CPU:** Intel Core i7 / AMD Ryzen 7 以上
    *   コア数が多いモデルがデータの前処理などで有利です。
*   **メモリ:** 32GB以上 (64GB以上を推奨)
    *   大規模なデータセットを扱う場合にメモリ容量が重要になります。
*   **ストレージ:** 1TB以上のNVMe SSD
*   **GPU:** NVIDIA GeForce RTX 4060 (VRAM 8GB) 以上
    *   機械学習フレームワークの多くがNVIDIAのCUDAを利用するため、NVIDIA製GPUが推奨されます。VRAM容量も重要です。

### ゲーム開発・3D CAD向け

*   **CPU:** Intel Core i7 / AMD Ryzen 7 以上
    *   コンパイルやビルド、3Dレンダリングの時間を短縮できます。
*   **メモリ:** 32GB以上
*   **ストレージ:** 1TB以上のNVMe SSD
    *   ゲームエンジンやアセットデータは容量が大きいため、大容量かつ高速なストレージが求められます。
*   **GPU:** NVIDIA GeForce RTX 4070 / AMD Radeon RX 7700 XT 以上
    *   リアルタイムレンダリングや複雑な3Dモデルを快適に扱うために、高いグラフィック性能が必要です。
